# How to install and run

1. Clone repo

<pre>
    <code>
       git clone 
    </code>
</pre>

2. Install npm packages command

<pre>
    <code>
       npm install
    </code>
</pre>

3. Run Application command

<pre>
    <code>
        npm run dev
    </code>
</pre>

Run `npm run dev` for a dev server. 
Local server `http://localhost:4200/`
Mock API Server `http://localhost:4201/`
